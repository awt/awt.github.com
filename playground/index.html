Playground

Playground is a WebKit based browser that watches a project directory for changes to files based on extension type. Whenver a change is detected it reloads the page via localhost on a configurable port. The Web Inspector is enabled by default for easy javascript debugging.

Playground is great for developers using a web framework working on javascript intensive pages, html, or css.
